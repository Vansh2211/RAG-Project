## 📌 RAG Project

This project implements a **Retrieval-Augmented Generation (RAG)** pipeline that processes PDF documents and answers user queries using a Large Language Model. It loads and splits documents into chunks, generates embeddings, retrieves the most relevant context, and produces accurate, context-aware responses using the Groq API. The system is modular, efficient, and designed as a foundation for building real-world AI applications like document assistants and knowledge-based chatbots.
